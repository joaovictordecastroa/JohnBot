# JohnBot
API para consumo do bot John criado para o teste da Take para a vaga de desenvolvimento. Foram criados dois projetos para a resolução do desafio, um em JavaScript com Node.js e outro em Java utilizando o Spring Boot. Para a integração com o Bot, foi necessário o deploy das aplicações no Heroku. 

Os projetos estão na mesma versão do Git.
